How It Works

  1. The owner creates a guest access key for you
  2. They send you a QR code (via text, email, etc.)
  3. Scan the QR code with this interface
  4. The system verifies your key on the blockchain
  5. If valid, access is granted!
Secure & Trustless
All verification happens on the Algorand blockchain. No one can fake or forge access keys.
